# Save Image as Type

Chrome extension to save any image as PNG, JPG, WebP, AVIF, SVG, GIF, PDF, TIFF, or BMP — directly from the right-click context menu.

## Install
1. Clone or download this repo
2. Go to `chrome://extensions`
3. Enable **Developer mode**
4. Click **Load unpacked** → select this folder

## Formats
PNG · JPG · WebP · AVIF · SVG · GIF · PDF · TIFF · BMP
